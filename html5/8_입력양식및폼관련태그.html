form 태그 내의 input 요소들에 사용자가 입력한 정보들을
서버로 넘기면서 요청하는 여할을 수행하는 태그이다.
form actoin="" method=""
<form actoin="" method="">
* actoin 속성 : 폼 내에 입력된 값들을 전달할 서버에 대한 경로 제시
* method 속성 : 요청전송 방식을 지정하는 속성 get(기본값)/post > get 방식 : 요청시 사용자가 입력한 값들이 url에 노출되는 방식 (전달값에 크기 제한)
> post 방식 : 요청시 사용자가 입력한 값들이 url에 노출이 안되는 방식

체크박스와 라디오 버튼 관련 input 태그들

라디오 버튼 (type="radio")

- 제시한 여러개의 값들 중에서 한개만 선택해야만 할 때

- 선택된 값(checked)만 넘어감

- 선택된 값을 넘기고자 한다면 선택했을 때의 어떤값을 넘길건지 value속성을 이용해서 명시해야함
