배경 이미지 테스트

배경색 뿐만 아니라 배경 이미지를 넣을 수 있다.

      선택자 {
        background-image:url("이미지 경로");
        background-repeat: repeat|repeat-x|repeat-y|no-repeat;
        background-size: auto | contain | cover | px px|% %;
        background-position:left,right,center top,bottom,center|px px|% %;
        background-attachment: scoll| fixed;
      }
