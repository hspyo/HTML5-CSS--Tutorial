텍스트 스타일

color 스타일속성 : 텍스트의 색상을 지정할 때 사용

        선택자{
            color:색상명|16진수|rgb|rgba|hsl|hsla;
        }
  • 색상명으로 지정
  • 16진수로 지정
  • rgb로 지정
  • rgba로 지정
  • hsl로 지정
  • hsla로 지정

text-decoration 스타일속성 : 텍스트에 줄을 긋거나 줄을 없앨 때 사용

        선택자{
            text-decoration:none|underline|overline|line-through;
        }
  • 텍스트 영역 아래 줄 긋기
  • 텍스트 영역 중간에 줄 긋기
  • 텍스트 영역 위에 줄 긋기
네이버로 이동

text-transform 속성 : 영문 텍스트의 대소문자 변환시 사용

        선택자{
            text-transform:none|uppercase|lowercase|capitalize;
        }
  • uppercase : 모든 영문자 다 대문자로
  • LowerCase : 모든 영문자 다 소문자로
  • capitalize : 영문자의 firsttext만 대문자로

text-align 스타일속성 : 텍스트 정렬할 때 사용

        선택자{
            text-align:left|right|center|justify;
        }
