3. 동위 선택자

동위 관계(같은 레벨)에 있는 뒤에 위치한 특정요소를 선택할 때 사용

      * a요소 뒤에 있는 b요소 하나만을 선택
      a+b{
        스타일속성: 값;
      }
      
      * a요소 뒤에 있는 모든 b요소 선택
      a~b{
        스타일속성: 값;
      }

4. 반응선택자

사용자의 움직임에 따라 달라지는 선택자

      * 사용자가 해당 요소 클릭했을 때 선택
      선택자:active{
        스타일속성:값;
      }

      * 사용자가 해당 요소에 마우스를 올려다놨을 때 선택자
      선택자:hover{
        스타일속성:값;
      }
